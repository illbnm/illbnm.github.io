1.命名空间

2.c++三目运算符

可以当左值,因为返回的是引用

3.const不能被修改

c++对const不能修改用指针修改的只是一个临时的temp

４．引用的特点

int &re = a ；（定义时出现&为引用 ，re 为a的别名，并且不分配内存 ）
re = b ;(已经声明，不可改变)
int &re ；error （定义引用必须初始化）
int &re3 = re； （可对引用再次引用)

5.对常量的引用

引用的是常量的临时空间变量的别名
